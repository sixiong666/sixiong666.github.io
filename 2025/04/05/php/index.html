基础语法

框架

  • <?php开头
  • ?>结尾
  • 每个语句之间必须用;隔开

变量

变量前面必须要加上$符号

作用域

  • 变量的作用域是脚本中变量可被引用/使用的位置。

  • php有三种作用域

    • local
    • global
    • static
    • 通常，当函数完成/执行时，其所有变量都会被删除。但是，有时我们希望局部变量不被删除。我们需要它来完成进一步的工作。
    • static就可以使局部变量不被删除

输出

echo print

区别：

echo 可以输出多个数据，而print只能输出一个

echo在输出多个数据时，类型相同使用.分隔，不同则使用,分隔

var_dump

这是用来输出变量的相关信息的函数，看一下变量的数据类型是什么

####数组

数组是一种特殊的变量，可以在单个名称下保存多个值，您可以通过引用索引号或名称来访问这些值。

通过**array**来进行创建

php的数组有三种类型

索引数组

关联数组

多维数组

  • 索引数组
    • 在索引数组中，每个项目都有一个索引号。
    • 默认情况下，第一个项目的索引是 0，第二个项目的索引是 1，依此类推。
  • 关联数组
    • 关联数组是通过键值对的形式进行存储的
    • 可以通过相对应的键值来访问改数组的值
  • 多维数组
    • 包含一个或多个数组的数组，没啥好说的
    • 下面这个就是外层是索引数组，内层放了很多的关联数组

函数

除了 PHP 内置函数之外，还可以创建自己的函数。

  • 函数是程序中可以重复使用的一组语句块。
  • 页面加载时，函数不会自动执行。
  • 函数将通过调用函数来执行。

关键字 function 开头，后跟函数名

控制结构

分支结构

if…else..

  • if 语句 - 如果某个条件为真，则执行一些代码
  • if...else 语句在条件为真时执行一些代码，而在条件为假时执行另一段代码。
  • if...elseif...else 语句 - 对两个以上的条件执行不同的代码
  • if语句也是可以进行简写的，可以将 if 语句写在一行上。

switch语句

  • 使用 switch 语句来选择要执行的多个代码块之一
  • 工作原理
    • 表达式只计算一次
    • 将表达式的值与每个 case 的值进行比较
    • 如果找到匹配项，则执行相关联的代码块
    • break 关键字会跳出 switch 块
    • 如果没有匹配项，则执行 default 代码块

注意：break的作用是停止该语句，如果没有break的话，会引起穿透的，简单点说就是他从匹配成功的语句下方的代码全部执行一遍

循环结构

  • 循环结构总共有四种，while,do...while,for循环,foreach循环
  • 最常用的是for循环，do....while 基本很少用，了解即可
  • foreach是专门用来遍历数组的，对数组的打印非常友好

while循环

  • while 循环会在指定的条件为真时执行代码块。
  • 用来写无限循环一般都爱用while

注意：记得增加条件$i的值，不然就会陷入到死循环里面。

do…while循环

  • 无论条件是什么，该循环必然会执行一次
  • do...while先执行一次代码，再来判断条件是否成立，为真则继续执行

for循环

语法：

  • test1为起始条件，只会执行一次
  • test2为终止条件，一旦条件不符合，则不在执行该循环
  • test3则是每次循环迭代后执行的语句，可以控制循环次数

foreach循环

  • foreach 循环最常见的用途是循环遍历数组的元素
  1. 索引数组的遍历
  • 每次迭代之后，当前数组$x的值会传给$$xx，一直到迭代完最后一个元素终止
  1. 关联函数的遍历
  • 和索引数组的遍历基本一致，就是关联数组的元素是由键值对组成，所以他的数据需要分别传入到$k和$v中

魔术方法

魔术方法有很多，每一个都是以--开头的

__construct()、 __destruct()、 __call()、 __callStatic()、 __get()、 __set()、 __isset()、 __unset()、 __sleep()、 __wakeup()、 __serialize()、 __unserialize()、 __toString()、 __invoke()、 __set_state()、 __clone()、 __debugInfo()

这里只讲几种

**注意：*除了 __construct()、 __destruct() 和 __clone() 之外的所有魔术方法都必须*声明为 public， 否则会发出 E_WARNING。 在 PHP 8.0.0 之前没有为魔术方法 __sleep()、__wakeup()、__serialize()、__unserialize()、__set_state() 发出诊断信息

构造函数

  • __construct
  • PHP 允许开发者在一个类中定义一个方法作为构造函数。具有构造函数的类会在每次创建新对象时先调用此方法，所以非常适合在使用对象之前做一些初始化工作。

扩展

$_SERVER

  • $_SERVER 是 PHP 中的一个超全局变量，用于存储关于头信息、路径和脚本位置的信息。
元素/代码 描述
$_SERVER[‘PHP_SELF’] 返回当前执行脚本的文件名。
$_SERVER[‘GATEWAY_INTERFACE’] 返回服务器使用的 Common Gateway Interface (CGI) 的版本。
$_SERVER[‘SERVER_ADDR’] 返回主机服务器的 IP 地址。
$_SERVER[‘SERVER_NAME’] 返回主机服务器的名称（如 www.w3school.com.cn）。
$_SERVER[‘SERVER_SOFTWARE’] 返回服务器标识字符串（如 Apache/2.2.24）。
$_SERVER[‘SERVER_PROTOCOL’] 返回信息协议的名称和修订号（如 HTTP/1.1）。
$_SERVER[‘REQUEST_METHOD’] 返回用于访问页面的请求方法（如 POST）。
$_SERVER[‘REQUEST_TIME’] 返回请求开始的时间戳（如 1377687496）。
$_SERVER[‘QUERY_STRING’] 如果页面是通过查询字符串访问的，则返回查询字符串。
$_SERVER[‘HTTP_ACCEPT’] 返回当前请求的 Accept 头。
$_SERVER[‘HTTP_ACCEPT_CHARSET’] 返回当前请求的 Accept_Charset 头（如 utf-8,ISO-8859-1）。
$_SERVER[‘HTTP_HOST’] 返回当前请求的 Host 头。
$_SERVER[‘HTTP_REFERER’] 返回当前页面的完整 URL（不可靠，因为不是所有用户代理都支持它）。
$_SERVER[‘HTTPS’] 脚本是否通过安全的 HTTP 协议查询。
$_SERVER[‘REMOTE_ADDR’] 返回用户查看当前页面的 IP 地址。
$_SERVER[‘REMOTE_HOST’] 返回用户查看当前页面的主机名。
$_SERVER[‘REMOTE_PORT’] 返回用户在机器上用于与 Web 服务器通信的端口。
$_SERVER[‘SCRIPT_FILENAME’] 返回当前执行脚本的绝对路径名。
$_SERVER[‘SERVER_ADMIN’] 返回在 Web 服务器配置文件中为 SERVER_ADMIN 指令给定的值。
$_SERVER[‘SERVER_PORT’] 返回服务器上 Web 服务器用于通信的端口（如 80）。
$_SERVER[‘SERVER_SIGNATURE’] 返回添加到服务器生成页面的服务器版本和虚拟主机名。
$_SERVER[‘PATH_TRANSLATED’] 返回当前脚本基于文件系统的路径。
$_SERVER[‘SCRIPT_NAME’] 返回当前脚本的路径。
$_SERVER[‘SCRIPT_URI’] 返回当前页面的 URI。
